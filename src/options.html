8rain Station Enhancer Options

Play Bookmarks

Play bookmarks are extra navigation links shown on the plays and bet market details page. They are intended to be custom plays filtering links. They can be added on the plays and bet market details pages and they can be reordered and deleted here.

Text Link

Sportsbook Link Target

Sportsbook link target controls how clicks on sportsbook names are launched into tabs. The "new tab" option will cause clicks to open into new tabs. The "book tab" option will cause a search for an existing tab for the book, and will only create a new tab if an existing tab is not found.

Book Details

Book details controls the mapping between book names seen in the app and the URL they link to. It also controls which tabs will get closed when using the enhancer to close open sportsbooks.
