8rain Station Enhancer Options

Book Details

Book details controls the mapping between book names seen in the app and the URL they link to. It also controls which tabs will get closed when using the enhancer to close open sportsbooks.

Active Books Names

Active books names represent a list of active books that can be loaded on the "Books" page. They can be created and modified from the Books page and can be deleted here.

Active Book Weightings Names

Active book weightings names represent a list of book to weightings mappings that can be loaded on the "Weightings" page. They can be created and modified from the Weightings page and can be deleted here.
